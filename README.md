## Dashboards de vendas de XBOX

Criei um projeto de dashboard no Excel, referente à vendas da Microsoft dos produtos Xbox, vinculando assinaturas da EA Pass e Minecraft Pass.
Coloquei filtros, que demonstram os valores anuais, mensais e trimestrais, deixando somente a planilha de dashboard visível, ocultando base e os cálculos ocultos.
