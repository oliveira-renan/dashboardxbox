# dashboardxbox
Criei um projeto de dashboard no Excel, referente à vendas da Microsoft dos produtos Xbox, 
vinculando assinaturas da EA Pass e Minecraft Pass.

No qual coloquei filtros, que demonstram os valores anuais, mensais e trimestrais.

Deixei somente a planilha de deashboard, deixando a base e os cálculos ocultos.
